💱 Conversor de Moeda
Aplicativo web simples para conversão de valores entre diferentes moedas. Desenvolvido como parte de um treinamento, utilizando HTML, CSS e JavaScript.

🚀 Tecnologias Utilizadas
HTML5

CSS3 (armazenado em Assets/style.css)

JavaScript (armazenado em Assets/script.js)

⚙️ Instalação e Execução
Clone o repositório: https://github.com/fabiogarcia-dev/Conversor-de-Moeda.git
bash
git clone 
Abra o arquivo index.html diretamente no navegador.
Não é necessário servidor ou dependências adicionais.

📖 Uso
Insira o valor desejado no campo de entrada.

Selecione a moeda de origem e a moeda de destino.

Clique em Converter para visualizar o resultado.

(Sugestão: incluir prints ou GIF mostrando a interface em ação)

✨ Funcionalidades
Conversão entre diferentes moedas.

Interface simples e responsiva.

Separação de responsabilidades:

HTML para estrutura

CSS em Assets/style.css para estilização

JavaScript em Assets/script.js para lógica da conversão
